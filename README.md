# DSP Production Solver

`DSP Production Solver` 是一个纯 vibe 项目。

它目前是一个面向《戴森球计划》的静态 Web 工作台和求解器原型：加载数据集，编辑目标与约束，然后在浏览器里查看配方方案、物品平衡和功耗结果。当前内置数据集只有 `Vanilla` 和 `OrbitalRing`。

项目仍然是探索式的，结构和语义都可能继续调整。
